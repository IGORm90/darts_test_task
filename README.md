# Задача:

Реализовать систему с возможностью создания товаров и наборов из товаров с пользовательским интерфейсом.
Пользовательским интерфейсом должны быть предусмотрены возможности для создания товара, удаления товара, создания набора, удаления набора, вывода списка товаров и наборов.
Товар должен содержать информацию о названии и стоимости.
Набор может состоять из товаров и других наборов. При этом, в конкретном наборе не может быть одинаковых товаров и наборов, но может быть в поднаборе (вложенном наборе). Товар или набор не может быть удалён, пока он состоит в каком-нибудь наборе.
Список товаров и наборов в пользовательском интерфейсе должен включать название товара/набора и стоимость (суммарную) товара/набора.
Также необходимо предусмотреть сохранение состояния системы (продуктов и наборов) любым известным способом.
